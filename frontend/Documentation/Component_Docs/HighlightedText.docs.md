# HighlightedText Component Documentation

## Overview

The `HighlightedText` component is designed to display text with highlighted sentences based on AI-generated or human-written classification. It also includes an optional legend to explain the highlighting colors.

## Properties

The `HighlightedText` component accepts the following properties:

- **text**: The plain text to display. If not provided, the component will use the `sentences` prop to construct the text.
- **sentences**: An array of sentence objects, each containing the sentence text and its classification result.
- **showLegend**: A boolean indicating whether to show the legend explaining the highlighting colors. Default is `true`.
- **className**: An optional class name to apply to the component.

## Usage

### HighlightedText with Sentence Analysis

```jsx
import HighlightedText from './HighlightedText';

const sampleSentences = [
  {
    sentence: "This is a human-written sentence.",
    result: {
      classification: "Human-Written",
      confidence: 0.92
    }
  },
  {
    sentence: "This appears to be generated by AI.",
    result: {
      classification: "AI-Generated",
      confidence: 0.87
    }
  },
  {
    sentence: "Another human written example here.",
    result: {
      classification: "Human-Written",
      confidence: 0.95
    }
  }
];

<HighlightedText sentences={sampleSentences} showLegend={true} />
```

### Plain Text without Analysis

```jsx
import HighlightedText from './HighlightedText';

<HighlightedText
  text="This is just plain text without any sentence analysis data."
  showLegend={false}
/>
```

### Empty Data

```jsx
import HighlightedText from './HighlightedText';

<HighlightedText
  text=""
  sentences={[]}
  showLegend={true}
/>
```

## CSS Styling

The `HighlightedText` component uses the following CSS classes:

- `.highlighted-text`: The main container for the component.
- `.highlighted-text-container`: The container for the text and legend.
- `.highlighted-text-content`: The area where the actual text is displayed.
- `.sentence-highlight`: The base style for each highlighted sentence.
- `.sentence-highlight.ai-generated`: The style for AI-generated text.
- `.sentence-highlight.human-written`: The style for human-written text.
- `.text-legend`: The container for the legend.
- `.legend-item`: Each item in the legend.
- `.legend-color`: The color boxes in the legend.
- `.legend-color.ai-generated`: The color box for AI-generated text.
- `.legend-color.human-written`: The color box for human-written text.
